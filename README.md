Instruções para execução do código.

Passo 1: No sheel interativo do Octave execute Main

- Existe ainda a possibilidade de executar as rotinas de Ordenação apenas:

1 - Para executar o InsertionSort, utilize: vetorOrdenado = InsertionSort(vetor);
	onde vetor é o vetor que se deseja ordenar.
 
2 - Para executar o QuickSort, utilize: vetorOrdenado = QuickSort(vetor, 1, n);
	onde vetor é o vetor que se deseja ordenar e n o número de elementos do vetor.
 
3 - Para executar o RandomizedQuickSort, utilize: vetorOrdenado = RandomizedQuickSort(vetor, 1, n);
	onde vetor é o vetor que se deseja ordenar e n o número de elementos do vetor.

4 - Para executar o MergeSort, utilize: vetorOrdenado = MergeSort(vetor, n);
	onde vetor é o vetor que se deseja ordenar e n o número de elementos do vetor.
